Default 'Flow' layout algorithm is used to create documents, which is why headers are blocks and text and links are inline. Grow or shrink, extra space distributed, etc. Flexibility is the goal.

Apparently grid is now common in all browsers and is also great. But flexbox is dynamic and fluid, which can be useful in certain cases. Depends on the kind of site you want.
